## **资源配置**

资源配置，执行下面可以进行相应的配置。

![资源配置](/themes/daux/img/1/4-2.png)

具体的内容包括：
1. 资源名称
2. 配置规格
3. 节点数量
4. 购买时长
5. 如有链克抵扣券，优先扣除抵扣券
6. 自动延长服务，勾选自动延长服务，系统会自动扣除链克中心的链克数量。
